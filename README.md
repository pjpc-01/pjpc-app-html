# 📱 手机NFC签到系统

这是一个纯手机端的NFC签到系统，可以直接在手机浏览器中使用。

## 🚀 访问地址

**GitHub Pages**: https://pjpc-01.github.io/pjpc-app-html/

## 📋 使用说明

1. 在手机浏览器中访问上述地址
2. 确保使用HTTPS连接
3. 点击"NFC签到"按钮
4. 将NFC卡片靠近手机背面
5. 等待读取成功提示

## 🔧 系统要求

- **Android**: Chrome 89+ 或支持Web NFC API的浏览器
- **iOS**: Safari 14+ (iOS 13+)
- **网络**: 需要互联网连接访问DDNS服务器

## 🔗 连接信息

- **PocketBase服务器**: http://pjpc.tplinkdns.com:8090
- **数据同步**: 自动发送到远程PocketBase服务器

---

**这是一个完全独立的手机端系统，不会影响你的电脑！** 🎉
