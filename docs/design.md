# PJPC Design Rules

## Icons
- **全部用 Lucide React 图标**，禁止 emoji 或自定义 SVG
- 示例：❌ 🏠 → ✅ `Building2` / `Home` | ❌ ⚡ → ✅ `Zap` | ❌ 🔗 → ✅ `Link`
- 图标大小统一：标题 `h-5 w-5` / 列表项 `h-4 w-4` / 按钮 `h-3.5 w-3.5`

## Colors
- 主色：blue-600
- 小学 (Primary)：blue 系
- 中学 (Secondary)：green 系
- 水费：blue-500
- 电费：yellow-500 / amber-500
- 金额支出：red-600 / orange-600
- 金额收入：green-600

## Typography
- 全站中文
- 数字金额用 `font-mono` / `tabular-nums`
